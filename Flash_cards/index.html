Variables

Variables

In JavaScript, it's conventional to give variables names that start with a lowercase letter, and if they are more than one word, to capitalize the first letter of subsequent words. This is called lower camel case.

Parameters

Parameters

In a function when an argument is assigned to a variable, the variable is called a parameter.

Functions

Functions

A function is something that performs an action. Just like a method, a function can take an argument. The alert() function pops up a dialog box with the string that you pass in as an argument.

Comments

Comments

In JavaScript, everything after the // is a comment, and is ignored. Comments are a convenient way to leave notes in your code for yourself or other programmers.
